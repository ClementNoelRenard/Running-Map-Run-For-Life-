Evolue sur la carte avec ta position gps, atteint les objectifs de parcours et obtient la chance d'echapper à une orde de zombies qui parcours les rues.

Version d'android compatible : 8 à 16 / besoin du gps en permanence. internet au debut (2-3s) pour charger la map. app essayé sur mon pixel 10 :D
